# お絵描きソフトの拡張

大学3年の授業「情報メディア実験A」で制作した、Java製のデスクトップアプリケーションです。
様々な図形を描画して、装飾をしたりアニメーションをつけることができます。

## 📦 概要
このアプリケーションは、マウス操作で簡単に図形を描けるツールです。影を付けたり枠線の太さを変える、図形の色を変えるなどの装飾も可能です。コピーペーストをする際には、Windowsの場合Ctrl+Cで選択した図形をコピーし、「貼り付け」を押してから任意の箇所をクリックすることでその場所に図形がペーストされます。図形を選択する際には、「選択」を押してから選択したい図形をクリックします。複数選択にも対応しており、「選択」を押してからマウスをドラッグすることで選択する範囲を指定できます。保存する際は左上の「ファイル」から保存してください。画像を挿入したい場合は、左上の「画像」から挿入したい画像を選択してください。「きゅうくらりん」は複製機能です。「きゅうくらりん」を押してから図形をクリックすると、その図形の横に同様の図形が複数複製されます。「テトリス」は回転アニメーションです。「テトリス」を押してから図形をクリックすると、その図形が回転します。回転した図形をもう一度クリックすることで、回転を止めることができます。

## 🎮 実行方法（推奨）
JavaがインストールされているPCであれば、コンパイルなしですぐに動かすことができます。
### 1. 事前準備
このアプリを動かすには **Java** が必要です。
もしPCに入っていない場合は、以下からインストールしてください。
- [Javaのダウンロード](https://www.java.com/ja/download/)

### 2. ダウンロード
右側の **Releases** ページ（または以下のリンク）から、最新の `.jar` ファイルをダウンロードしてください。
- [ダウンロードはこちら (Releases)](https://github.com/matsu1122/javaGUI/releases/tag/v1.0)

### 3. 起動
ダウンロードした `.jar` ファイルをダブルクリックすると起動します。
